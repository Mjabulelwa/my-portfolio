### 6. [Sales Forecasting Dashboard System]<a href="https://github.com/Mjabulelwa/my-portfolio/blob/main/forecasting_system_dashboard.py">Sales Forecasting Dashboard System</a>
This Sales Forecasting Dashboard is an interactive application built with Streamlit that allows users to upload sales data, analyze historical performance, and generate future sales predictions. The system processes the data by calculating total sales, visualizing daily trends, and applying a time series forecasting model using SARIMAX. It enables users to adjust the forecast period, view projected sales with confidence intervals, and explore results through clear charts and tables, helping support better planning and decision-making.

### 7. [Doctor Booking Dashboard System]<a href="https://github.com/Mjabulelwa/my-portfolio/blob/main/doctor_dashboard.py">Doctor Booking Dashboard System</a>
Doctor Appointment Booking Dashboard is an interactive healthcare application built with Streamlit that allows users to schedule, manage, and analyze doctor appointments in real time. The system enables users to input doctor details, select specialties and available time slots, and book appointments for patients through a simple sidebar interface.

Once bookings are made, the dashboard dynamically stores and displays the data in a structured table, along with key performance indicators such as total bookings and the number of unique patients. It provides both interactive and analytical visualizations using Plotly and Seaborn, including bar charts, pie charts, count plots, and heatmaps. These visuals help users identify trends such as the busiest doctors, most popular time slots, and distribution of medical specialties.
